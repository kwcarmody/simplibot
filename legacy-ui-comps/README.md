# Legacy UI Components

This folder contains the older static SPA-style frontend artifacts that are no longer the active runtime for SimpliBot.

These files are being retained as a **style and UI reference** only.

## Not the active app
The current live implementation is the Express + EJS app described in `docs/README.md` and `docs/architecture.md`.

- `js/`

Use this folder for visual/reference ideas, not as the primary implementation path.
